KMBuilder

case class KMBuilder(km: KernelMessage = ...) extends Product with Serializable

A class for building KernelMessages. Upon creation, holds a KernelMessage with empty parameters. A new KMBuilder holding a KernelMessage with modified parameters can be generated using the withXYZ() methods. The KernelMessage held by the KMBuilder can be obtained with build(). All metadata returned by metadataDefaults() is added to the KernelMessage's metadata when build() is called.

e.g. val base = KMBuilder().withParent(parentMessage)

val km1 = base.withHeader(MessageType.ExecuteReply).build

val km2 = base.withHeader(MessageType.ExecuteRequest) .withContentString("content").build
